# Weldix




## Stack

- Frontend: React + Vite + Tailwind CSS + Framer Motion
- Backend: FastAPI + SQLAlchemy + JWT
- Base de datos: SQLite (configurable por `DATABASE_URL`)
- Testing: Pytest

## Vision de Weldix

Este repositorio es la base de una solucion para procesos internos: control de usuarios, permisos por rol y futura expansion a modulos de operacion, trazabilidad y administracion.
